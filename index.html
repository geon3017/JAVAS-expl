    부족한점

    • 1. 메인 페이지에서 사진들 하나씩 fade in / fade out 할 수 있게 구현을 못 한게 부족함
    • 2. 서브페이지는 새창으로 열리는게 편하다고 생각을 했는데 내가 사이트를 이용이 불편해서 수정
    • 3. README의 파일의 작성 부분에서 팀원이 읽기 쉽게 작성을 했어야 하는 아쉬움
    • 4. 같은 파일을 여러명이 수정했을 때 충돌하는 부분이 있어서 이걸 해결하는게 어려움
    • 5. 아직은 디테일한 부분은 많이 부족한거 같다 좀 더 꼼꼼하게 확인을 못한점

    잘한점

    • 팀원들과의 적극적으로 소통
    • github를 이용해서 README파일로 그날에 무엇을 했는지 작성
    • ㄴ 그날 작업을 일지로 작성하니깐 내가 다음에 작업을 할때 무엇을 해야 하는지 알 수 있어서 편리했다.
    • 잘모르는 스크립트는 구글에 검색을 해서 나한테 맞는 구조로 만들어서 활용
    • green sock의 플러그인을 사용해서 이벤트를 적용

    배운점

    • 1. siwper적용
    • 2. 외부 자바스크립트 적용
    • 3. github를 이용한 README파일 활용과 git 활용
    • 4. github에서 파일 충돌시 해결 방법
    • 5. 팀 프로젝트의 중요성
    • 6. 구글검색으로 자바스크립 활용

    개인 리뷰

    • 팀 프로젝트 하면서 어려운 점은 없었다,
    • 다들 너무 잘해주셨고 소통도 빠르고 활발해서 작업이 편했다, 다들 각자 자기가 맡은 부분을 잘 해주셔서 잘 마무리
      가 되었고 부족한 부분은 서로가 잘 채워줘서 css나 자바스크립 할 때 도움이 많이 됐다.READMDE 파일을 활용이
      많이 도움이 된 거 같다, 언제 어디 부분에서 수정을 했는지 한눈에 볼 수 있어서 좋았다.
    • 주석을 좀 더 세세하게 나눠서 작성한게 도움이 됐다, 전에는 큰 부분만 나눠서 주석을 작성을 했는데 다른 팀원이
      세세하게 적은 주석을 봤는데 너무 잘 정리가 되어있어서 그 이후에 세세하게 적으려고 노력하고 있고 주석 활용을
      어떻게 하면 좋은지를 배운거 같다.
